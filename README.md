# Unity Interaction Design Framework
*A powerful, extensible, generic, and versatile interaction design framework for building complex puzzle systems.*

**Status:** In active development

**Version:** 0.0.1

## Components
* **Logic Gates**
	* And Gate
	* Nand Gate
	* Or Gate
	* Nor Gate
	* Xor Gate
	* Xnor Gate
	* Inverter
* **Buttons**
	* Click Button
	* Hold Button
	* Mash Button
	* Push Button
* **Switches**
	* Click Switch
	* Hold Switch
	* Mash Switch
	* Flick Switch
	* Sensor Switch
* **Sensors**
	* Stateless Sensor
	* Stateful Sensor
* **Selectors**
	* Selector
	* Weighted Selector
	* **State Selectors**
		* Bistate Selector
		* Quadstate Selector
		* Octostate Selector
* **Sequencer**
	* Sequencer
	* Chance Sequencer
	* Random Sequencer
* **Misc**
	* Distance Checker
	* Counter
	* Timer
	* Metronome
